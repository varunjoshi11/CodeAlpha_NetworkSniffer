### 1. Network Sniffer
A Python-based network traffic analyzer using Scapy and Npcap. Captures live packets and provides detailed protocol analysis.

#### Features
- Real-time packet capture with source/destination IP, MAC addresses, protocols (TCP/UDP/ICMP)
- TCP details: sequence/ACK numbers, flags, TTL
- Handles fragmented IP packets and Ethernet frames
- Packet length, capture time, and more

#### Quick Start
pip install scapy
python advanced_sniffer.py
**Note**: Requires Npcap on Windows for packet capture.
